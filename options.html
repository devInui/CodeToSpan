CodeToSpan Options

Exclude Tags:

Select tags where <code> elements will not be converted to <span> tags.

Apply Language-Based Control

When enabled, the extension will only apply modifications if the page's language differs from your browser's language.

Skip Styled Code Tags

Skips replacing <code> tags styled as block elements to prevent layout disruptions due to changes in text width from translations.

Add translate="no" to

Automatically add a "no translate" attribute to <pre> tags to prevent translation of code blocks without code tags.

Exclude Domains

    Add domains to exclude from this extension's processing. Recommended for sites that generate <code> tags automatically, such as generative AI.
    This extension is active, you can view the current page's domain in the developer console.
